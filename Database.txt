3. CHƠI VỚI EF-CORE (ENTITY FRAMEWORK CORE)
*EF CORE LÀ BỘ THƯ VIỆN LẬP TRÌNH TRONG .NET, GỒM NHIỀU FILE .DLL VÀ .EXE ĐỂ GIÚP DÂN DEV CHƠI VỚI TABLE, NƠSQL MÀ KO CẦN VIẾT CÁC CÂU LỆNH SQL TRUYỀN THỐNG, HẬU TRƯỜNG SẼ LO GIÚP MÌNH, MÌNH CHỈ CHẤM GỌI HÀM THÔI

*EF CORE GỒM 2 CỤM CÔNG CỤ
- NHỮNG FILE .DLL ĐỂ CUNG CẤP CÁC CLASS CODE, ĐỂ GIÚP DÂN DEV TA CODE VỚI TABLE
- TA DOWLOAD NHỮNG FILE .DLL NÀY QUA CÔNG CỤ TÊN LÀ NUGET
		(JAVA CÓ TOOL TƯƠNG ĐƯƠNG TÊN LÀ: MAVEN)
		(NODEJS, REACT CÓ TOOL TƯƠNG ĐƯƠNG TIẾP LÀ NPM)
		(PYTHON CÓ TOOL TƯƠNG ĐƯƠNG TÊN LÀ PIP)
